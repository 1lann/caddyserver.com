User Guide

These documents describe how to use, configure, and extend Caddy.

If you're new to Caddy, try the Getting Started guide.

Once you know how to run Caddy, be sure check out the Caddyfile so you can configure your site and get the most out of Caddy.

The rest of these documents are organized by directive name. You can learn how to use Caddy's many features by looking at the documentation for the various directives.

For Developers

Did you know you can use Caddy middlewares in your own Go application? Check out the developer guide on the GitHub project wiki.
