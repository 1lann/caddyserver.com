redir

redir sends the client an HTTP redirect status code if the URL matches the specified pattern.

Syntax

redir from to code
  • from is the request path to match (it must match exactly)
  • to is the path to redirect to
  • code is the HTTP status code to respond with; must be in the range [300-308] excluding 306. May also be meta to issue meta tag redirect.
redir to [code]
  • to is the path to redirect to
  • code is the HTTP status code to respond with, as above; default is 301 Moved Permanently

Omitting the from argument creates a "catch-all" redirect: any request will be redirected to the destination while preserving the URL's suffix.

Examples

When a request comes in for /images/photo.jpg, redirect to /images/drawing.jpg with a HTTP 301 (Moved Permanently) status code:

redir /images/photo.jpg /images/drawing.jpg 301

When any request comes in, redirect to https://mysite.com with a HTTP 301 Moved Permanently status code:

redir https://mysite.com

Redirect any requests to /under-construction.html with a 307 Temporary Redirect status code.

redir /under-construction.html 307
