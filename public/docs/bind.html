bind

bind overrides the host to which the server should bind. Normally, the server binds to the IP address to which the hostname resolves. For example, if your Caddyfile is for mysite.com, Caddy will look up the host mysite.com to know the IP address to bind to. However, that does not work well in all environments, so you can force Caddy to try binding to another IP address instead.

Syntax

bind host
  • host is the hostname (or IP address) to bind to

Examples

To bind to IP 0.0.0.0 (wildcard host) no matter what the hostname is in the server address:

bind 0.0.0.0
